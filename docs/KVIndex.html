KVIndex Documentation

A thread-safe, process-safe key-value store that mimics Python's dict interface with ACID properties and supports various eviction policies including LRU, LFU, FIFO, and random.

Initialize

from liteindex import KVIndex
kv_index = KVIndex(db_path="./test.liteindex")

Parameters

  • db_path: Path to the index file. Defaults to None, which will create an in-memory index.
  • store_key: Defaults to True. If set to False, index can only be used for lookup.
  • preserve_order: Defaults to True. If set to False, insert/update order is not preserved.
  • ram_cache_mb: Size of the RAM cache in MB. Defaults to 32.
  • eviction: Eviction policy. Defaults to EvictionCfg(EvictionCfg.EvictNone).

Insert/Update single or multiple

kv_index["key1"] = "value1"
kv_index.update({"key2": "value2", "key3": "value3"})

Get single or multiple

kv_index["key1"]
kv_index.get("key1", "default_value")
kv_index.getvalues(["key1", "key2"])

Delete single or multiple, clear

del kv_index["key1"]
kv_index.delete(["key1", "key2"])

kv_index.clear()

Iteration and reverse iteration

for key in kv_index: pass

for key in kv_index.keys(reverse=True): pass
for value in kv_index.values(reverse=True): pass
for key, value in kv_index.items(reverse=True): pass

len, contains

len(kv_index)
"key1" in kv_index

EvictionCFG

Configure the eviction policy using the EvictionCfg class.

from liteindex import EvictionCfg
eviction_cfg = EvictionCfg(
    eviction_policy=EvictionCfg.EvictNone, # Eviction policies: EvictNone, EvictLRU, EvictLFU, EvictAny
    max_size_in_mb=0, # No size-based eviction
    max_number_of_items=0, # No count-based eviction
    invalidate_after_seconds=0, # No time-based eviction
)
